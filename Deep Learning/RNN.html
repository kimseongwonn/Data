RNN

Recurrent Neural Networks → ‘Recurrent’ : 재귀성, 반복되는 → 시간에 따라 가중치(w)가 반복되고 사용된다

  • 가중치와 활성화 함수의 관계
    • 가중치란?
      • 모델의 학습 과정에서 매우 중요한 역할을 하는 파라미터
      • 신경망의 각 뉴런 간 연결의 강도를 나타내며 입력 데이터에서 출력으로의 변환을 조정

      1. 가중치의 역할

      • 가중치는 입력 신호를 변환하여 출력으로 전달하는 데 사용
      • 신경망의 기본 단위인 뉴런은 다음과 같은 방식으로 작동
        • 각 입력 값 x_i에 가중치 w_i를 곱합니다.
        • 이 값들을 모두 더한 다음, 바이어스(bias) b를 더합니다.
        • 그 결과에 활성화 함수(Activation Function)를 적용하여 최종 출력을 생성합니다.

      수식으로 표현하면:

      2. 가중치 초기화

      • 가중치의 초기화는 신경망의 학습 과정에 큰 영향을 미칩니다. 초기화 방법은 여러 가지가 있으며, 대표적인 방법은 다음과 같습니다:
        • 무작위 초기화 (Random Initialization): 가중치를 작은 무작위 값으로 초기화합니다. 이는 대칭성 문제를 피하고 학습을 시작할 수 있게 합니다.
        • He 초기화: ReLU 활성화 함수를 사용하는 신경망에서 주로 사용되며, 가중치를 평균이 0이고 분산이 2/n인 정규 분포에서 초기화합니다.
        • Xavier 초기화 (Glorot 초기화): 시그모이드나 tanh 같은 활성화 함수를 사용하는 신경망에서 주로 사용되며, 가중치를 평균이 0이고 분산이 2/(n_in + n_out)인 정규 분포에서 초기화합니다. 여기서 n_in은 입력 뉴런의 수, n_out은 출력 뉴런의 수입니다.

      3. 가중치 학습

      • 신경망의 학습 과정에서 가중치는 훈련 데이터를 통해 조정
      • 이는 주로 역전파 알고리즘(Backpropagation)을 통해 이루어짐
        • 순전파 (Forward Propagation): 입력 데이터를 네트워크에 통과시켜 출력을 계산합니다.
        • 손실 계산 (Loss Calculation): 예측된 출력과 실제 값을 비교하여 손실(오차)을 계산합니다.
        • 역전파 (Backpropagation): 손실을 미분하여 각 가중치에 대한 그래디언트를 계산합니다.
        • 가중치 업데이트 (Weight Update): 그래디언트 디센트(Gradient Descent) 등의 최적화 알고리즘을 사용하여 가중치를 업데이트합니다.

      가중치 업데이트 수식은 다음과 같습니다

      4. 최적화 방법

      • 경사하강법 (Gradient Descent): 가장 기본적인 방법으로, 미분을 사용하여 손실을 최소화하는 방향으로 가중치를 조정합니다.
      • 모멘텀 (Momentum): 이전 기울기 값도 고려하여 가중치를 업데이트함으로써 수렴 속도를 높이고 진동을 줄입니다.
      • Adam (Adaptive Moment Estimation): 학습률을 적응적으로 조정하며, 모멘텀과 RMSProp을 결합한 방법입니다.

      5. 가중치가 신경망 성능에 미치는 영향

      • 초기화: 적절한 초기화가 없으면 신경망이 학습을 시작하기 어렵거나 매우 느리게 학습될 수 있습니다.
      • 학습률: 학습률이 너무 크면 발산할 수 있고, 너무 작으면 수렴 속도가 느려질 수 있습니다.
      • 과적합/과소적합: 가중치가 학습 데이터에 너무 맞춰지면 과적합(overfitting)이 발생할 수 있고, 반대로 너무 일반적이면 과소적합(underfitting)이 발생할 수 있습니다.

      요약

      • 가중치(Weights): 신경망에서 입력과 출력 간의 연결 강도를 나타내는 파라미터.
      • 초기화: 무작위 초기화, He 초기화, Xavier 초기화 등.
      • 학습: 순전파, 손실 계산, 역전파, 가중치 업데이트.
      • 최적화 방법: 경사하강법, 모멘텀, Adam 등.
      • 성능 영향: 초기화, 학습률, 과적합/과소적합 등.
  • 시계열 데이터(시간에 따라 변화하는 데이터)를 처리하기에 좋은 뉴럴 네트워크 구조
    • 음성인식(speech recognition)
    • 음악 생성기(music generation)
    • DNA 염기서열
    • 번역기(machine translation)
    • 감정 분석(sentiment classification)
    • → 자연어 처리, 음성 처리 등등 모두 순차성을 가진다 : 시간에 흐름에 따라 진행된다
      • 예시) 나는 물을 마시고 싶어 → ‘마시고’라는 단어는 ‘나는’, ’물을’이 순차적으로 나오고 나와야 문장에서 의미를 가진다. 즉, 시간의 흐름에 따라 정보가 나열된다.

First Order System

  • 현재 시간의 상태가 이전 시간의 상태와 관련이 있다고 가정
  • 자기와 자기 이전 상태에 대한 함수
  • 즉 t일 때 x는 이전 시간인 t-1일 때 x에 영향을 받는다(단, 이전 시간의 상태만 영향을 미친다 → x0는 x1에 영향을 미치고 x1은 x2에 영향을 미친다 즉, x0가 x2에 명향을 미치지는 않는다.)
  • 현재 시간의 상태는 이전 시간의 상태에 대한 어떤 함수이다
  • 이 시스템은 외부 입력 없이 자기 혼자서 돌아간다 → autonomous system

현재의 입력이 추가된 First Order System

State-Space Model

  • 1차원 시스템의 모형
  • 모든 시간 t에서 모든 상태 xt가 관측 가능한가?(observable)
    • 날씨 예측
    • 주가 예측
    • 예시) 날씨를 예측하는 것이 가능한가? → 일부만 가능하다 → 날씨에 영향을 주는 다양한 요인들이 존재하는데 그 요인 중 일부만 예측할 수 있기 때문에 일부만 가능하다고 할 수 있다
  • 즉, 모든 시간 t 에서 모든 상태 xt는 일부만 관측 가능하다 → 관측 가능한 상태의 모음(State-Space Model에서 정의된다)
  • 각 시간에서 관측 가능한 상태의 모음 : 출력 yt
  • State-Space Model as RNN
    • 상태 x_t가 의미하는 것은 hidden layer의 state이다
    • 이전가지의 상태와 이전까지의 입력을 대표할 수 있는 압출본이라고 할 수 있음
      • 예시) 오늘의 주가는 어제까지의 주가를 결정한 과거의 주가와 주가에 영향을 주는 변수들의 입력값이 압축된 총체하고 할 수 있음
    • 상태 x_t는 시계열로 들어오는 입력들을 최대한 상세히 표현할 수 있어야함

어떤 시스템을 해석하기 위한 3요소

  • 입력(u), 상태(x), 출력(y)
  • 도식화 → 입력값(x), 상태값(s), 출력값(o)
    • 그림에서 알 수 있듯 단순히 입력값 → 상태값 → 출력값 단계를 거치는 ANN과 달리 self-loop 단계가 존재해서 일정한 W(가중치)가 반복 사용된다

RNN에서의 가중치(Weight)

위의 이미지 1을 참고

  • 입력 가중치 행렬 : 입력 데이터 x_t를 현재 은닉 상태 s_t로 변환하는 가중치 → U(W_xs)
  • 은닉 상태 가중치 행렬 : 이전 은닉 상태 s_t-1을 현재 은닉 상태 s_t로 변환하는 가중치 → W(W_ss)
  • 출력 가중치 행렬 : 현재 은닉 상태 s_t를 출력 o_t로 변환하는 가중치 → V(Wso)

RNN 작동 원리

  • 은닉 상태 업데이트
  • 출력 계산
  • 초기 조건 정의(s_0) : RNN이 진행됨에 있어 첫번째 시점의 상태는 따로 정의해주어야 한다

RNN의 유형

  • 일대일 : RNN이 없는 기본 처리 모드, 고정된 크기의 입력에서 고정된 크기의 출력으로 진행 (예 - 이미지 분류)
  • 일대다 : 시퀀스 출력(예 - 이미지를 가져와 이미지를 설명하는 단어와 문장을 출력 > CNN으로 생성한 이미지 특징 벡터를 사용하여 RNN의 초기 은닉 상태로 사용) → 생성
  • 다대일 : 시퀀스 입력(예 - 주어진 문장이 긍정적 또는 부정적인 감정을 표현하는 것으로 분류되는 감정 분석) → 예측
  • 다대다(1) : 시퀀스 입력 및 시퀀스 출력(예 - 번역기로 영어 문장을 프랑스 문장으로 번역) → 번역
  • 다대다(2) : 동기화된 시퀀스 입력 및 출력(예 - 비디오의 각 프레임에 레이블을 지정하려는 비디오 분류)

RNN 한계점

1. 기울기 소실 문제 (Vanishing Gradient Problem)

  • 설명: 역전파 과정에서 기울기(Gradient)가 점점 작아져서 초기 층까지 도달하기 전에 거의 0이 되는 현상입니다. 이는 특히 긴 시퀀스를 학습할 때 심각한 문제입니다.
  • 영향: 모델이 장기 의존성을 학습하지 못하고, 최근의 정보에만 의존하게 됩니다.
  • 원인: RNN에서 은닉 상태는 반복적으로 이전 은닉 상태에 가중치를 곱하는 방식으로 업데이트됩니다. 이 과정에서 활성화 함수의 미분 값이 1보다 작으면, 기울기가 반복적으로 곱해지면서 기하급수적으로 감소합니다.

2. 기울기 폭발 문제 (Exploding Gradient Problem)

  • 설명: 기울기가 역전파 과정에서 너무 커지는 현상입니다.
  • 영향: 모델의 가중치가 비정상적으로 커지며, 학습이 불안정해지고 발산할 수 있습니다.
  • 원인: 기울기가 1보다 큰 값이 반복적으로 곱해질 때, 기울기가 기하급수적으로 증가하게 됩니다.

3. 장기 의존성 학습의 어려움

  • 설명: 긴 시퀀스에서 초기 입력이 나중의 출력에 영향을 미치는 경우, 이를 학습하는 것이 어렵습니다.
  • 영향: RNN은 단기 의존성을 잘 학습할 수 있지만, 장기 의존성은 학습하기 어렵습니다.
  • 원인: 기울기 소실 및 폭발 문제로 인해 장기적인 정보가 효과적으로 전달되지 못합니다.

4. 병렬 처리의 어려움

  • 설명: RNN은 순차적으로 데이터를 처리하기 때문에 병렬 처리가 어렵습니다.
  • 영향: 학습 속도가 느려지며, 특히 긴 시퀀스를 처리할 때 시간이 많이 소요됩니다.
  • 원인: RNN의 특성상 다음 타임 스텝의 은닉 상태는 이전 타임 스텝의 은닉 상태에 의존하므로, 모든 타임 스텝을 동시에 처리할 수 없습니다.

5. 계산 비용

  • 설명: 긴 시퀀스를 학습하고 예측하는 데 높은 계산 비용이 소요됩니다.
  • 영향: 모델을 학습시키고 예측하는 데 시간이 많이 걸리며, 자원이 많이 필요합니다.
  • 원인: 각 타임 스텝마다 은닉 상태를 계산하고 업데이트해야 하기 때문입니다.

6. 제한된 메모리 용량

  • 설명: RNN의 은닉 상태는 고정된 크기이기 때문에, 긴 시퀀스의 모든 정보를 효과적으로 저장하고 활용하기 어렵습니다.
  • 영향: 긴 시퀀스에서 중요한 정보를 잃어버릴 수 있습니다.
  • 원인: 은닉 상태의 크기가 고정되어 있으며, 이는 모델의 메모리 용량을 제한합니다.

RNN의 문제점 추가 학습

  • RNN은 아래 두 가지 원인으로 시계열 데이터의 장기 의존 관계를 학습하기 어렵다는 문제점 존재
    • 기울기 소실
    • 기울기 폭발
  • RNN의 경우 가중치 조정이 모델의 성능에 아주 중요한 역할을 함
  • 따라서 가중치를 반복적으로 조정하면서 결과값과 실제값의 차이를 줄이는 것이 중요
  • 이때 사용하는 방식이 바로 역전파(Back propagation) 방식
    • 역전파(Back propagation)
      • 인공 신경망을 통해서 도출한 결과값과 실제값을 비교하여 Loss(손실)과 관련된 손실함수를 구하고 역으로 출력 → 입력 방향으로 손실이 적어지는 방향으로 가중치를 업데이트 하는 과정
      • 역전파 과정
        1. 순전파 (Forward Propagation)
          • 입력 데이터를 신경망에 주입하고 각 층에서 가중치를 적용하여 출력을 계산
          • 각 층에서는 활성화 함수를 통해 비선형성을 추가하고 다음 층으로 신호 전달
        1. 손실 함수 계산
          • 손실 함수(loss function)란 신경망 모델이 예측한 출력과 실제 타겟 값 사이의 차이를 측정하는 함수
          • 손실 함수 값을 최소화하는 것이 모델의 성능을 높이는 것이므로 이를 위해 가중치를 조정
        1. 역전파 (Backward Propagation)
          • 손실 함수를 사용하여 각 가중치가 전체 오차에 얼마나 기여했는지 계산
          • 이 과정에서 역전파는 출력층에서부터 시작하여 각 측의 가중치에 대한 오차의 그라디언트(gradient - 기울기)를 계산
          • 그라디언트는 출력층에서부터 역순으로 각 층의 가중치와 편향을 조정하기 위해 사용

        4. 가중치 업데이트

        • 계산된 그라디언트를 사용하여 경사 하강법을 이용해 가중치를 업데이트
        • 경사 하강법은 손실 함수를 최소화하기 위해 가중치를 조금씩 조정하는 방법
        • 가중치를 업데이트 할 때 가중치에 대한 오차의 기울기(gradient)를 계산하여 곱해준다
        • 이 때 기울기 소실 문제가 발생
        • tanh(hyperbolic tangent) 함수 예시
          • RNN에서 일반적으로 쓰이는 tanh 함수는 다음과 같이 [-1,1] 사이의 범위의 값을 가짐
          • tanh 함수를 미분하게 되면 보라색과 같은 그래프가 나옴
          • 보라색 그래프의 최대값은 1이므로 1보다 작은 값이 오차의 기울기가 됨
          • 따라서 레이어가 많을 때 1보다 작은 값을 무수히 곱하게되면 값이 0으로 수렴하여 기울기가 손실되는 문제가 발생
