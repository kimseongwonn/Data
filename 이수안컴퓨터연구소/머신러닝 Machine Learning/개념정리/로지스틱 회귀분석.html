로지스틱 회귀분석

Logistic Regression

기본 개념

  • 선형 회귀 : y = wx + intercept
  • 다중 선형 회귀 : y = w1x1 + w2x2+…+wnxn + intercept
  • 선형회귀의 경우 선형 관계에 있는 두가지 이상의 변수를 가지고 독립변수(x)와 종속변수(y)를 정해서 독립변수들로 종속변수를 예측
  • 예시) 차량 가격, 디자인, 연비, 마력, 제조사 등의 차량 관련 데이터가 있을 때 각 특성들은 모두 선형적인 관계에 있다고 가정, 이 때 차량 가격을 종속변수(y)로 설정하고 나머지를 독립변수(x)들로 설정한 뒤 각 독립변수들의 가중치(w)와 회귀식의 절편(intercept)를 구해서 종속변수(y)인 차량 가격을 구하는 것이 선형 회귀분석
  • 로지스틱 회귀분석은 독립변수로 하여금 종속변수를 예측한다는 점에서 회귀분석이지만 예측하는 종속변수가 선형회귀와 달리 이산형(주로 범주형) 종속변수임
  • 예시) 쥐의 몸무게로 쥐가 비만인지 아닌지 예측하는 것은 로지스틱 회귀분석

로지스틱 함수

  • 선형회귀의 경우 ‘y = wx+c’와 같은 함수를 사용하여 종속 변수를 예측함
  • 그렇다면 로지스틱 회귀는 어떤 함수를 사용해야 비선형적 예측을 할 수 있을까?
  • 로지스틱 함수는 시그모이드 함수의 일종으로 함수식 도출 과정은 아래와 같다
    • 오즈(odds) : 성공 확율이 실패 확률에 비해 몇 배 더 높은가?
      • 예시) 6개의 빨간공과 4개의 파란공이 들어있는 바구니에서 빨간공이 뽑힐 확률은 60% 안뽑힐 확률은 40%일때 오즈는 [60/40=1.5]이다
    • 로짓 변환 : 오즈를 자연로그로 변환시키는 과정
      • 오즈를 로짓 변환시키는 이유
        1. 선형화 : 로짓 변환은 확률의 비선형성을 제거하고 선형 모델을 사용할 수 있게 함

          → 로지스틱 회귀에서 확률을 예측할 때 선형 회귀의 장점을 활용하고자 함 이때 확률은 0과1 사이의 값이기 때문에 선형 모델로 다루기 어려워 장점 활용이 어려움

          → 그러나 오즈는 0에서 무한대까지의 값을 가지므로 이를 로그 변환하여 값의 범위를 -∞에서 +∞까지 확장시킴으로 선형 회귀의 장점을 활용할 수 있도록 함

        1. 대칭성 : 로짓 변환은 확률의 양쪽 극단을 대칭적으로 다루게 함

          →오즈를 로그 변환 하면 결과 값이 대칭적으로 나타남

    • 로지스틱 함수 : 로짓 변환의 역함수로 해석 가능 / 선형 회귀(w^TX)와 sigmoid함수의 결합이다
  • 예시)

    로지스틱 회귀 모델의 선형 결합 부분을 간단한 데이터 예시를 통해 설명하겠습니다.
