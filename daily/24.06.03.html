    • 유방암 분류
      import numpy as np 
import pandas as pd 

from sklearn.datasets import load_breast_cancer

cancer = load_breast_cancer()
      • cancer.data 와 cancer['data'] 차이점

        cancer.data와 cancer['data']는 실제로 동일한 데이터를 참조하지만, 이 두 방식의 접근에는 약간의 차이가 있습니다. 그 차이를 이해하기 위해서는 load_breast_cancer 함수의 반환 객체가 무엇인지 알아야 합니다.

        load_breast_cancer의 반환 객체

        load_breast_cancer 함수는 sklearn.datasets 모듈에서 제공하는 데이터셋 중 하나로, 이 함수는 Bunch 객체를 반환합니다. Bunch 객체는 Python의 사전(dictionary)과 유사한 객체로, 키를 통해 값에 접근할 수 있는 특성이 있습니다. 또한, 속성(attribute) 방식으로도 값에 접근할 수 있습니다.

        cancer.data vs cancer['data']

        • cancer.data는 Bunch 객체의 속성(attribute) 방식으로 데이터에 접근하는 방법입니다.
        • cancer['data']는 Bunch 객체의 키(key) 방식으로 데이터에 접근하는 방법입니다.

        이 두 접근 방식의 차이점은 다음과 같습니다:

        1. 속성 접근 (cancer.data)
          • 이 방식은 점(.) 연산자를 사용하여 속성에 접근합니다.
          • 코드가 더 간결하고 직관적입니다.
          • 그러나, Python의 예약어와 충돌할 수 있으며, 속성 이름이 동적으로 결정될 경우에는 사용하기 어렵습니다.
        1. 키 접근 (cancer['data'])
          • 이 방식은 키(key) 연산자를 사용하여 값에 접근합니다.
          • 사전(dictionary) 스타일의 접근 방식으로, 키가 문자열로 제공됩니다.
          • 더 일반적이며, 동적으로 키를 생성하거나 접근할 때 유용합니다.
          • 예약어와 충돌하지 않습니다.

        예시 코드

        import numpy as np
import pandas as pd
from sklearn.datasets import load_breast_cancer

cancer = load_breast_cancer()

# 속성 방식 접근
data_attr = cancer.data
print(data_attr[:5])  # 처음 5개의 샘플 출력

# 키 방식 접근
data_key = cancer['data']
print(data_key[:5])  # 처음 5개의 샘플 출력

# 두 방식이 동일한지 확인
print(np.array_equal(data_attr, data_key))  # True 출력

        위 코드를 실행해보면 cancer.data와 cancer['data']가 동일한 데이터를 참조하고 있음을 확인할 수 있습니다.

        정리

        • 속성 방식 (cancer.data): 더 간결하고 직관적이지만, 예약어와 충돌할 수 있습니다.
        • 키 방식 (cancer['data']): 더 일반적이며 유연성이 높습니다. 예약어와 충돌하지 않습니다.

        이 두 방식은 Bunch 객체에서 데이터를 접근하는 방법일 뿐, 참조하는 데이터 자체에는 차이가 없습니다. 상황에 따라 두 방식 중 적합한 것을 선택해서 사용하면 됩니다.

  • 프로젝트 관련
    • 정규화, 표준화
      • 머신 러닝에서의 정규화와 표준화

        정규화와 표준화는 머신러닝에서 매우 중요한 데이터 전처리 기법입니다. 이 두 기법은 데이터의 스케일(크기)을 조정하여 모델의 성능을 향상시키고, 학습을 더 빠르고 안정적으로 진행할 수 있게 해줍니다.

        정규화 (Normalization)

        정규화는 데이터의 범위를 [0, 1] 또는 [-1, 1]과 같은 특정 범위로 조정하는 과정입니다. 일반적으로 최소-최대 정규화(Min-Max Normalization)를 사용합니다.

        수식

        $$ X_{norm} = \frac{X - X_{min}}{X_{max} - X_{min}} $$

        여기서:

        • \( X \)는 원본 데이터 값
        • \( X_{min} \)는 해당 특성의 최소값
        • \( X_{max} \)는 해당 특성의 최대값

        사용 이유

        • 균등한 기여: 데이터의 각 특성이 모델 학습에 균등하게 기여하게 합니다. 특히 K-최근접 이웃(KNN), 인공 신경망(ANN) 등의 거리 기반 알고리즘에서 중요합니다.
        • 수렴 속도 향상: 경사하강법을 사용하는 알고리즘의 수렴 속도를 향상시킵니다.

        예시

        데이터:
        \[ [10, 20, 30, 40, 50] \]

        정규화:
        \[ [0, 0.25, 0.5, 0.75, 1] \]

        표준화 (Standardization)

        표준화는 데이터의 평균을 0, 표준 편차를 1로 맞추는 과정입니다. 이는 데이터의 분포를 정규 분포(가우시안 분포)에 가깝게 만듭니다.

        수식

        $$ X_{std} = \frac{X - \mu}{\sigma} $$

        여기서:

        • \( X \)는 원본 데이터 값
        • \( \mu \)는 해당 특성의 평균값
        • \( \sigma \)는 해당 특성의 표준 편차

        사용 이유

        • 데이터 분포 조정: 데이터의 분포를 표준 정규 분포로 맞춰줍니다. 특히, 선형 회귀, 로지스틱 회귀, 서포트 벡터 머신(SVM) 등에서 중요합니다.
        • 모델 안정성: 데이터의 값이 너무 크거나 작을 때 발생할 수 있는 수치적 불안정을 줄입니다.

        예시

        데이터:
        \[ [10, 20, 30, 40, 50] \]

        평균:
        \[ \mu = 30 \]

        표준 편차:
        \[ \sigma = 15.81 \]

        표준화:
        \[ [-1.26, -0.63, 0, 0.63, 1.26] \]

        왜 사용해야 하는가?

        1. 모델 학습의 효율성: 스케일이 다른 특성들을 같은 스케일로 맞추면, 경사 하강법의 수렴 속도가 빨라지고, 최적화가 더 잘 됩니다.
        1. 모델 성능 향상: 정규화와 표준화를 통해 모델이 데이터의 스케일 차이로 인한 왜곡 없이 학습할 수 있게 됩니다.
        1. 비교 용이성: 특성 간의 상대적 중요도를 정확히 비교할 수 있습니다.

        실습 예시

        사이킷런을 사용하여 정규화와 표준화를 하는 예시입니다.

        import numpy as np
from sklearn.preprocessing import MinMaxScaler, StandardScaler

# 예시 데이터
data = np.array([[10, 2000], [20, 3000], [30, 4000], [40, 5000], [50, 6000]])

# 정규화
min_max_scaler = MinMaxScaler()
data_normalized = min_max_scaler.fit_transform(data)

print("정규화된 데이터:")
print(data_normalized)

# 표준화
standard_scaler = StandardScaler()
data_standardized = standard_scaler.fit_transform(data)

print("표준화된 데이터:")
print(data_standardized)

        이 코드를 실행하면, 각각의 특성에 대해 정규화와 표준화를 적용한 결과를 볼 수 있습니다. 이를 통해 각 특성의 스케일을 동일하게 맞출 수 있습니다.

        정리

        • 정규화: 데이터의 범위를 일정하게 맞춰줌으로써, 모델이 각 특성의 영향을 균등하게 받을 수 있도록 합니다.
        • 표준화: 데이터의 평균을 0, 표준 편차를 1로 맞춰줌으로써, 특성 간의 분포를 동일하게 만들어 모델의 성능을 향상시킵니다.

        정규화와 표준화는 데이터의 스케일을 조정하여 머신러닝 모델이 더 빠르고 정확하게 학습할 수 있도록 돕는 중요한 기법입니다.
