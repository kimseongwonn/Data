    • streamlit 공부

      Streamlit은 파이썬을 이용해 데이터 애플리케이션을 쉽게 만들 수 있는 오픈 소스 라이브러리입니다. Streamlit을 이용하면 웹 애플리케이션을 작성하는 데 필요한 HTML, CSS, JavaScript를 몰라도 간단하게 대화형 웹 앱을 만들 수 있습니다. 이제 Streamlit을 배우기 위한 단계별 가이드를 제공하겠습니다.

      1. 기본 개념 이해하기

      Streamlit의 특징

      • 간단한 API: 데이터 시각화와 대화형 위젯을 쉽게 만들 수 있는 직관적인 API를 제공합니다.
      • 실시간 업데이트: 코드 변경 시 실시간으로 웹 애플리케이션이 업데이트됩니다.
      • 데이터 시각화 라이브러리 통합: Pandas, Matplotlib, Plotly 등 다양한 데이터 시각화 라이브러리와 통합됩니다.

      2. 설치

      Streamlit을 사용하려면 먼저 Python이 설치되어 있어야 합니다. 그 후, pip을 사용해 Streamlit을 설치할 수 있습니다.

      pip install streamlit

      3. 기본 앱 만들기

      Streamlit 앱은 Python 스크립트입니다. streamlit run 명령어를 통해 실행합니다. 기본 앱 예제를 작성해 보겠습니다.

      # app.py
import streamlit as st

st.title('Hello, Streamlit!')
st.write('This is a simple Streamlit app.')

      이 스크립트를 실행하려면 터미널에서 다음 명령어를 입력합니다.

      streamlit run app.py

      4. 주요 기능 사용하기

      텍스트와 데이터 표시

      Streamlit은 다양한 텍스트와 데이터를 표시할 수 있는 기능을 제공합니다.

      import streamlit as st
import pandas as pd

st.title('Streamlit Tutorial')

st.header('Header')
st.subheader('Subheader')
st.text('This is a text.')
st.markdown('**Markdown** is also *supported*.')

# DataFrame 표시
df = pd.DataFrame({
    'Column 1': [1, 2, 3, 4],
    'Column 2': [10, 20, 30, 40]
})
st.dataframe(df)
st.table(df)

      대화형 위젯

      Streamlit은 대화형 위젯을 제공하여 사용자와 상호작용할 수 있습니다.

      st.button('Click me')
st.checkbox('Check me')
st.radio('Choose one', ['Option 1', 'Option 2'])
st.selectbox('Select one', ['Option A', 'Option B'])
st.slider('Slide me', 0, 100)

      시각화

      Streamlit은 다양한 시각화 라이브러리를 지원합니다.

      import matplotlib.pyplot as plt
import numpy as np

# Matplotlib 예제
x = np.linspace(0, 10, 100)
y = np.sin(x)

fig, ax = plt.subplots()
ax.plot(x, y)

st.pyplot(fig)

# Plotly 예제
import plotly.express as px

df = px.data.iris()
fig = px.scatter(df, x='sepal_width', y='sepal_length', color='species')
st.plotly_chart(fig)

      5. 앱 배포

      Streamlit Cloud를 이용해 앱을 배포할 수 있습니다. GitHub에 코드를 업로드한 후 Streamlit Cloud와 연동하여 쉽게 배포할 수 있습니다.

      Streamlit Cloud 사용 방법

      1. Streamlit Cloud 사이트에 접속해 회원가입 또는 로그인합니다.
      1. "New app" 버튼을 클릭하고 GitHub 리포지토리를 선택합니다.
      1. 배포하고자 하는 브랜치와 파일을 선택한 후 "Deploy" 버튼을 클릭합니다.

      이제 앱이 배포되고, 제공된 URL을 통해 접근할 수 있습니다.

      6. 실습 예제

      간단한 데이터 분석 애플리케이션을 만들어 보겠습니다.

      import streamlit as st
import pandas as pd
import plotly.express as px

# 타이틀
st.title('Simple Data Analysis App')

# 데이터 업로드
uploaded_file = st.file_uploader("Choose a file", type=['csv'])
if uploaded_file is not None:
    df = pd.read_csv(uploaded_file)

    # 데이터 프레임 표시
    st.write(df)

    # 기초 통계량 표시
    st.write(df.describe())

    # Plotly 시각화
    fig = px.histogram(df, x='column_name')
    st.plotly_chart(fig)

      위 코드는 사용자가 CSV 파일을 업로드하면 데이터프레임을 표시하고, 기초 통계량을 보여주며, 특정 열의 히스토그램을 Plotly로 시각화하는 간단한 데이터 분석 앱입니다.

      from openai import OpenAI

client = OpenAI(
    api_key = 'api key 넣는 곳'
)

image = 'https://upload.wikimedia.org/wikipedia/commons/thumb/d/dd/Gfp-wisconsin-madison-the-nature-boardwalk.jpg/2560px-Gfp-wisconsin-madison-the-nature-boardwalk.jpg'

query = '이 이미지를 묘사해봐'

response = client.chat.completions.create(
    model='gpt-4-vision-preview',
    messages =[
        {
         'role': 'user',
         'content': [
             {'type': 'text', 'text': query},
             {'type': 'image_url',
              'image_url': image
              },
             ],
        },
        ],
    max_tokens=300,
)

# respose에서 choice로 가서 0번째 요소로 가서 message로 가서 content를 뽑는다 >> 우리가 얻고자하는 묘사 문장
response.choices[0].message.content

# 두 이미지 비교
another_image = 'https://wikidocs.net/images/page/31379/transformer_from_encoder_to_decoder.PNG'

query = '두 장의 이미지에 대해 설명해줘. 어떤 차이가 있어?'

response = client.chat.completions.create(
    model='gpt-4-vision-preview',
    messages =[
        {
         'role': 'user',
         'content': [
             {'type': 'text', 'text': query},
             {'type': 'image_url',
              'image_url': image
              },
             {'type': 'image_url',
              'image_url': another_image
              },
             ],
        },
        ],
    max_tokens=1024, # max_token 길이 조정
)

print(response.choices[0].message.content)
